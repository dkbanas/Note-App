About

Welcome to the Note App, a CRUD (Create-Read-Update-Delete) web application designed for keeping your notes organized. Developed with cutting-edge technologies including .NET 8, Angular 18.1.2, and Bootstrap, this app ensures a smooth and efficient user experience.

Key Features

  • User Authentication: Secure login and registration using JWT Bearer authentication.
  • Personalized Notes: Each user has a private space to manage their notes.
  • Sorting and Searching: Easily sort notes by date or title, and use the search function to find specific notes.
  • CRUD Operations: Create, read, update, and delete notes effortlessly.

Navigation

  • Home: Welcome page with an overview of the app.
  • My Notes: Access your personal notes (authentication required).
  • About: Information about the project and its underlying technologies.
  • Account: View and update your account details.

Authentication

  • Login: Access your account by logging in.
  • Sign Up: Register for a new account.

Technologies Used

  • Frontend: Angular 18.1.2, Bootstrap
  • Backend: .NET 8, Entity Framework Core, PostgreSQL
  • Patterns: Clean Architecture, Repository Pattern

NuGet Addons

  • Microsoft.EntityFrameworkCore v8.0.7: Database context management
  • Npgsql.EntityFrameworkCore.PostgreSQL v8.0.4: PostgreSQL provider
  • Microsoft.EntityFrameworkCore.Design v8.0.7: Migration support
  • Microsoft.AspNetCore.Identity.EntityFrameworkCore v8.0.7: Identity management
  • Microsoft.IdentityModel.Tokens v8.0.1: Token handling
  • Microsoft.IdentityModel.JsonWebTokens v8.0.1: JWT support
  • Microsoft.AspNetCore.Authentication.JwtBearer v8.0.7: JWT Bearer authentication
  • Microsoft.Extensions.Identity.Stores v8.0.7: Core identity stores

Instructions

Using the App:

  1. Navigation: Use the top navbar to move between Home, My Notes, About, and Account pages.
  2. Authentication:
    • If not logged in, use the "Login" button to access your account or "Sign Up" to register.
    • If logged in, use the "Add Note" button to create a new note or "Logout" to sign out.
  3. My Notes:
    • View all your notes in the "My Notes" section.
    • Sort notes by date or title for easier management.
    • Use the search bar to quickly find specific notes.
    • Click on a note to view its details, edit, or delete it.
